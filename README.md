# Homebrew Tap - tvps

Homebrew tap for tvps tools and apps.

## Installation

```bash
brew tap if414013/tvps
```

## Available Formulas

| Formula | Description | Install |
|---------|-------------|---------|
| kiro-gateway | Proxy gateway for Kiro API - OpenAI and Anthropic compatible | `brew install if414013/tvps/kiro-gateway` |

## Usage

After tapping, install any formula:

```bash
brew install kiro-gateway
```
